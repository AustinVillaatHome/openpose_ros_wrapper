# openpose_ros_wrapper

HCRL version of openpose_ros_wrapper

You might need the latest version of Openpose (2018 FEB)

## run normal openpose_ros

```
rosrun openpose_wrapper openpose_wrapper_node
```

## run openpose_ros_3d

```
rosrun openpose_wrapper openpose_wrapper_node
```
